Carbon Footprint Logger Platform

A full-stack web application that allows users to log daily activities and track their carbon emissions. The system provides a simple dashboard to visualize environmental impact over time.


Project Description

The Carbon Footprint Logger Platform is designed to help users become more aware of their environmental impact by tracking daily activities such as transportation, electricity usage, and other emission-related actions.

Users can:
- Register and log into the system
- Add daily activity logs with emission values
- View a dashboard summarizing their carbon footprint
- Track their environmental impact over time

This project demonstrates full-stack development using a React frontend and a Node.js/Express backend.


Features

Authentication
- User registration system
- User login system


Carbon Tracking
- Add daily activities
- Store emissions data
- Automatically timestamp entries
- View all logged activities

Dashboard
- Displays all carbon logs
- Shows emission values per activity
- Updates dynamically after new entries

Full-Stack Integration
- React frontend
- Backend API
- RESTful communication between client and server


Tech Stack

Frontend
- React.js
- CSS
- HTML

Backend
- Node.js
- Express.js


Data Storage
- In-memory JavaScript arrays (temporary storage)
